⚠️ What "installing" actually costs
A normal installer copies your app into C:\Program Files, writes machine-wide registry keys, and often installs the .NET runtime for all users. All three live above the line — shared, Windows-protected areas — so Windows shows the UAC prompt and blocks anyone without administrator rights.
💡 Key question
Installing means crossing that admin line. So how can SPRecorder record meetings, save files, and even start on login — all from a ZIP — without ever asking for admin? What is it deliberately not doing?
📐 The trade-offs of being portable
  • 🆚 It isn't "installed": no Start Menu entry and no auto-update — you manage the folder yourself.
  • 📦 Bigger download: self-contained carries the whole .NET runtime (~255 files). That heft is the price of needing nothing machine-wide.
  • 🔁 Auto-start (still no admin): drop a shortcut into your shell:startup folder — that's inside your own profile too.
  • ⚠️ SmartScreen still warns once: that's about the app being unsigned (trust), not admin. Click "More info → Run anyway".
  • 🚫 Don't move it into Program Files: that location does need admin and would defeat the whole point.
