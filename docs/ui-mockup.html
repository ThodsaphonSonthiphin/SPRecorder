SPRecorder — UI mockup (tray-only app, no main window)

All visible UI lives in the Windows system tray. There is no main window — configuration is via appsettings.json. The mockup shows: (1) the tray icon in two states, (2) the right-click context menu in two states, and (3) the toast notifications the app raises.

Where you'll see it on screen: the gray/red square at the bottom-right of this page (in the fake taskbar) is exactly where SPRecorder lives in real Windows — next to the network/volume/clock icons. Click it to toggle in/out of "recording" state in this mockup.
